Web Audio API

    What is it?
  • High level JavaScript API meant to process and synthesize audio in web applications
  • Mostly used for interactive and dynamic applications like web games, web comics, and any other interactive media
    How does it work?
  • All audio is managed with an instance of AudioContext
  • One instance of AudioContext is needed per application, because it can manage one or many sound sources
  • A sound needs to be loaded into a buffer via an XMLHttpRequest and the decodeAudioData method of AudioContext
Click play to start the music, and stop to stop. If stopped and played again, the music will start from the beginning. The music takes a moment to start.
